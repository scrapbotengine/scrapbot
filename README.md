# Scrapbot Engine

An compact, experimental, and probably mostly useless game engine that tries to answer the question: what if a game engine was 100% agentically engineered, and structured specifically so agents can help you build your game?

## Features / Roadmap

### General

- [x] Single-binary executable, written in Zig
- [x] Cross-platform support for Windows, macOS, and Linux
- [x] Fully ECS-driven architecture with generation-aware entities, component tables, and system schedules

### Rendering

- [x] 3D rendering through wgpu-native
