# Machina Engine Contributing Guide

## This is an Agentic-First Codebase

PRs are welcome, but unlike many other projects, this codebase is designed to be primarily maintained with the help of coding agents. Significant work has been done to make the codebase agent-friendly (agentic rules, skills, guardrails, and more); please make use of this harness.

- Please use a capable coding agent.
- Please declare PRs that were primarily generated by a human as such.
- If you want to make changes to the harness itself, please make sure they're posted as a separate PR.
