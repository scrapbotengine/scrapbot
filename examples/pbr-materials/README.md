# PBR Materials

A deterministic reference scene for Scrapbot's authored material resources.

The upper row is dielectric (`metallic = 0`) and the lower row is metallic
(`metallic = 1`). Roughness increases from left to right:

`0.05`, `0.25`, `0.50`, `0.75`, `1.00`.

Run it with:

```sh
scrapbot run examples/pbr-materials
```
